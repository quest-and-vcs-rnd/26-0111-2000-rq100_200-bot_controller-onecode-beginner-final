## Use as Extension

This repository can be added as an **extension** in MakeCode.

* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/quest-and-vcs-rnd/26-0111-2000-rq100_200-bot_controller-onecode-beginner-final/releases/tag/v0.2.4** and import
